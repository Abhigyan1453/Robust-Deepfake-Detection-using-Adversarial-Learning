# Robust-Deepfake-Detection-using-Adversarial-Learning
This project focuses on leveraging Adversarial Learning techniques to enhance the robustness of Deepfake Detection systems against evolving manipulation methods.
## 🧠 Pre-trained Model & Weights
Trained model file (`model.pth`) GitHub ki file size limit se badi hai, isliye ise Google Drive par host kiya gaya hai. 

👉 **[Yahan se Model Weights download karein](https://drive.google.com/drive/folders/1eYpFt0WYKTgY2kMpvcAmaNPJRG2Idnzq?usp=sharing)**

> **Note:** Download karne ke baad `model.pth` file ko project ke main folder (root directory) mein rakhein taaki `app.py` use access kar sake.
